# cljs-format-converter

Application for data conversion.
Supported formats - JSON, YAML, EDN, JavaScript Object.

## Usefull commands
* Install deps
```
make install
```

* Build tailwind styles
```
make tw-build
```

* Run app
```
make dev
```

* Run tests
```
make test
```

* Release build
```
make release
```
